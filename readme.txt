FMxA is a 3D racing time trial game I made as part of my Cross-Platform Development subjects at AIE.

------------------------------------------------------------

DOCUMENTATION

Documentation is generated using doxygen. Documentation files are contained in Assets/Scripts/Documentation.


------------------------------------------------------------

HOW TO PLAY (PC)

Use WASD to move your car, hold L-SHIFT to use boost.

Be sure to slow around corners: Boost is only regained when going slow.

Press SPACE to use your handbrake.

------------------------------------------------------------
